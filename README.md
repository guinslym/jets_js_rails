## Installation

Add this line to your application's Gemfile:

```ruby
gem 'jets_js_rails'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jets_js_rails

Into Rails
```ruby
#application.js
//= require jets
```
